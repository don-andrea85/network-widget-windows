# Hacker News Draft

## Title

Show HN: Network Widget, a tiny Windows Wi-Fi/Ethernet switcher

## Body

I built a small open source Windows utility for switching Wi-Fi and Ethernet adapters without opening Windows Settings.

It is a compact desktop widget with two switches, green/red adapter states, tray support, resizing, startup shortcut creation, and an installer. The original use case was a home setup where Wi-Fi can be better for some downloads, while Ethernet is still useful for other workflows.

GitHub:
https://github.com/don-andrea85/network-widget-windows

Release:
https://github.com/don-andrea85/network-widget-windows/releases/latest

It is intentionally simple and written in C#/.NET Framework so it can run on Windows 10/11 without a heavy runtime.
