# Product Hunt Draft

## Name

Network Widget

## Tagline

Switch Wi-Fi and Ethernet from a tiny Windows desktop widget.

## Description

Network Widget is a small open source Windows utility that lets you enable or disable Wi-Fi and Ethernet adapters without opening Windows Settings.

It was built for setups where Wi-Fi may be faster for some downloads while Ethernet remains useful for stability or specific workflows. The widget is compact, movable, resizable, tray-friendly, DPI-aware, and includes a simple installer.

## First Comment

Hi Product Hunt,

I built Network Widget because I kept switching between Wi-Fi and Ethernet on Windows and did not want to dig through Settings every time.

It is intentionally small: two switches, clear adapter states, tray support, startup support, and an installer. Windows requires elevation for adapter changes, so the app can register optional scheduled tasks to reduce repeated UAC prompts after setup.

The project is open source on GitHub, and feedback is very welcome.
